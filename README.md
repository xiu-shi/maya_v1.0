# Maya - Janet's Digital Twin

Maya is Janet Xiu Shi's digital twin - a ChatGPT-like interface powered by AI Builders API.

## Quick Start

### Prerequisites
- Node.js >= 18.0.0
- AI Builders API token

### Setup
1. Clone repository
2. Copy `Maya/backend/.env.example` to `Maya/backend/.env`
3. Add your `AI_BUILDER_TOKEN` to `.env`
4. Install dependencies: `cd Maya/backend && npm install`
5. Start server: `npm start`

### Deployment
See `Maya/DEPLOYMENT.md` for deployment instructions.

## Project Structure

- `Maya/backend/` - Node.js/Express backend
- `Maya/frontend/` - HTML/CSS/JS frontend
- `Maya/tests/` - Test suite
- `Maya/knowledge/` - Knowledge base
